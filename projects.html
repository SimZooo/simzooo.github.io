netlens

(github)

A network visualization site for monitoring and analyzing low-level network traffic in real-time. Some technologies used include Svelte for the frontend and Rust for the backend packet capturing and processing. Features include dynamic graphs, detailed packet information, and customizable views. The networking was done using the low-level pnet library in Rust, with a HTTP server built using Actix-web to serve the frontend and provide real-time data via WebSockets.
